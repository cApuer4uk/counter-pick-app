Dota 2 Settings:

Resolution: 1920×1080

Display Mode: Borderless

Game Mode: Captain's Draft

After Installation:

Launch CounterPick Launcher from the Start Menu.

⚠️ IMPORTANT:
Run Counterpick.exe **as Administrator**.

Otherwise, the app will not be able to write JSON files
(overlay_data.json and overlay_state.json) inside Program Files.

Usage:
1. Open Dota 2 and wait until the draft screen appears.
2. Press START in the launcher.
3. Switch back to Dota 2 (ALT+TAB).
4. The application will display counter-picks as heroes are drafted.
